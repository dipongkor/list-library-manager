List/Library Manager

Following tasks can be done using this App very easily

  • Find List/Library
  • Add new List/Library
  • Edit List/Library Properties
  • Delete all items at a time
  • Add new column
  • Reorder column in forms
  • Delete column from List/Library
  • Delete List/Library

This App in a single page app. We hope you will have a nice UX. As this App is free, so please don't forget to rate this App.
